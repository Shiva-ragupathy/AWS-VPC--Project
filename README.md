# AWS-VPC--Project
Built a custom VPC in Amazon Web Services with CIDR 10.0.0.0/16, including public (10.0.1.0/24) and private (10.0.2.0/24) subnets. Configured an Internet Gateway and route table for internet access, and launched a Linux EC2 instance in the public subnet with secure SSH access.
